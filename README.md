# Smart Study Planner

## Project Overview
Smart Study Planner is a web-based application designed to help students manage their courses, assignments, and deadlines while automatically generating weekly study plans to improve productivity and time management.

## Features
- Course Management
- Assignment & Deadline Tracking
- Weekly Study Plan Generation
- Progress Tracking
- Notification System
