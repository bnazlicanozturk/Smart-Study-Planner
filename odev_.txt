

1.Executive summary
Introduction:Smart Study Planner is a web-based planning solution created through our group work.

Students may forget their assignments, schedules can be confusing, studying can become inefficient, and some students may not have enough storage space on their phones.

Solution: web-based planning system
Notification will be sent via reminder or SMS method
Technology:modern web technologies

2.Product description and vision 
Problem
Our problem: Students may forget their assignments, schedules can be confusing, studying can become inefficient, and some students may not have enough storage space on their phones.

Solution
Web-based
Email/SMS reminder
Adding courses
Creating tasks
Deadline tracking
Creating study schedules
Target audience: University students
Students/teachers with busy schedules
Individuals who cannot manage their time

For those who experience stress and panic due to a heavy course load and cannot manage their schedule

3. CORE Features
Actions that can be performed in the system:
User Registration and login; can create an account, register, log in, create an account with email and password
Adding courses,
Can add courses and edit their times,
Task management:
Users can create assignments and tasks
Weekly/monthly study planner;
Can create a weekly/monthly study plan
Reminder system:
Email/SMS is sent to the user upon request when the deadline approaches
Reminders can be sent when class times approach
When the required class time in the study plan approaches
Security: Users use a secure login system.

4. Technical Architecture
How the system works

Architecture
Client-Server Architecture
Layers
Frontend: The web interface that users see
Backend:
Where the logic of the work running in the background is executed
Data flow:
1. User adds tasks/assignments/lessons
2. Data is sent to the backend
3. Backend saves to the database
4. The system returns a success message to the user
