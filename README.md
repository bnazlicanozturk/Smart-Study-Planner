# Smart Study Planner

## Project Overview
Smart Study Planner is a web-based application designed to help students manage their courses, assignments, and deadlines while automatically generating weekly study plans. The main goal of the system is to improve time management and productivity among students.

## Features
- Course Management
- Assignment and Deadline Tracking
- Automatic Weekly Study Plan Generation
- Progress Tracking
- Notification System
